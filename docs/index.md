# wechat-ilink-bot 文档

欢迎使用 `wechat-ilink-bot`。

这是一个面向 WeChat iLink Bot API 的 Python SDK，目标是让你可以快速完成：

- 扫码登录并持久化账号
- 机器人长轮询收消息
- 主动发送文本/图片/视频/文件
- 通过 Webhook 让外部系统触发消息发送
